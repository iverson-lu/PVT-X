# Documentation Overview

This folder contains the **current authoritative documentation** for PVT-X.

## Current Documents
- pvtx-core-spec.md
  Engine, runner, execution model, and data contracts.

- pvtx-ui-spec.md
  UI structure, workflows, and design guidelines.

- pvtx-test-authoring_rules.md
  Unified authoring rules for all Test Cases in the PVT-X PC Test Suite.

- pvtx-id-schema.md
  Official 4-level identifier schema for PVT-X test cases.

## Archive
Documents under `docs/archive/` are **historical or experimental** and
MUST NOT be used as implementation references.
They are kept only for record purposes.
